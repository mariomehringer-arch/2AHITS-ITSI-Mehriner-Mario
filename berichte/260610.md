# Recherchiere und experimentiere – cut: in welchen Varianten können Felder mit der -f Option angegeben werden? was unterscheidet die -c Option von -f?

## in welchen Varianten können Felder mit der -f Option angegeben werden?

Felder können als einzelnes Feld, als Liste mehrerer Felder, als Bereich, vom ersten Feld bis zu einem bestimmten Feld, von einem bestimmten Feld bis zum Ende oder als Kombination dieser Varianten angegeben werden.


cut -f 2        # einzelnes Feld
```shh
~/workspace$ cut -d ',' -f 2 klassenkassa.csv | head
21.09.2023
22.09.2023
21.09.2023
18.09.2023
15.09.2023
22.09.2023
18.09.2023
20.10.2023
25.11.2023
05.11.2023
```
cut -f 1,3,5    # mehrere Felder
```shh
~/workspace$ cut -d ',' -f 1,3,5 klassenkassa.csv | head
Maximilian,3.0
Sophia,15.0
Jonas,3.0
Lena,13.0
Lukas,2.5
Emma,42.0
Nico,23.0
Jana,11.9
Timo,5.0
Laura,20.0
```
cut -f 2-4      # Bereich
cut -f -3       # vom ersten Feld bis Feld 3
cut -f 4-       # von Feld 4 bis zum Ende
cut -f 1,3-5    # Kombination

## was unterscheidet die -c Option von -f?
-c arbeitet mit Zeichenpositionen, -f mit Feldern

# Übung (Werkstatt Summe)
Schreibe ein shell Kommando (Einzeiler) das in klassenkassa.csv alle Beträge mit dem Text Werkstatt in folgender Form ausgibt:

3.0+15.0+3.0+13.0+2.5

Anleitung:

Verwende grep um die Zeilen mit Werkstatt zu filtern.
Verwende nicht die -c Option von grep.
wende cut auf das Ergebnis an um nur die Geld-Beträge zu filtern
Die Beträge stehen nun auf einzelnen Zeilen. Verwende das Tool paste um diese zu einer Zeile zusammenzufassen.

## Lösung
```shh
~/workspace$ grep "Werkstatt" klassenkassa.csv | cut -d ',' -f 3 | paste -sd '+'
15.0+13.0+42.0+23.0+33.2+5.3+25.0+3.21+15.78+18.64+6.51+7.54+3.21+15.78+18.64+6.51+6.75+5.67+10.56+12.67+15.21+5.78+14.89+18.76+12.54+14.32+12.54+13.76+6.78+8.54
```
